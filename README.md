# Vibraflux

## Project Overview
Vibraflux is designed to provide an efficient solution for managing vibration analysis and monitoring processes. It integrates advanced algorithms for data analysis and visualization, enabling users to easily interpret vibration data for predictive maintenance and quality assurance.

## Quick Start Guide
1. **Clone the Repository**:
   Run the following command to clone the repository:
   ```bash
   git clone https://github.com/leoncanales23/vibraflux.git
   ```

2. **Install Dependencies**:
   Navigate into the cloned directory and install the required dependencies:
   ```bash
   cd vibraflux
   pip install -r requirements.txt
   ```

3. **Run the Project**:
   Start the application by executing:
   ```bash
   python main.py
   ```

4. **Access the Interface**:
   Open your web browser and go to `http://localhost:8000` to access the interface.

For detailed documentation, please refer to the docs directory or visit our [Wiki](https://github.com/leoncanales23/vibraflux/wiki).
